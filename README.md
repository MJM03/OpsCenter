# Ops Center v1.5.0 PWA

## Subir a GitHub Pages
1. Descomprime el ZIP.
2. Sube todos los archivos y carpetas a la raíz del repositorio.
3. Activa GitHub Pages desde Settings > Pages.

## Instalar en iPhone
1. Abre la URL publicada en Safari.
2. Toca Compartir.
3. Selecciona Agregar a inicio.
4. Confirma con Agregar.

Si ya instalaste una versión anterior, elimínala de la pantalla de inicio y vuelve a instalarla para que aparezca el icono nuevo.
